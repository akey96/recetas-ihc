Comandos de voz

Para buscar receta > “Buscar [Nombre de la receta]”

Para borrar una busqueda > “Limpiar busqueda”

Para abrir siguiente hoja de resultados > "Pagina Siguiente"

Para abrir anterior hoja de resultados > "Pagina Anterior"

Para abrir una receta > "Abrir [Nombre de la receta]"

Para ir atras > “Atras”

Para abrir una receta por numero de Id de receta > "Abrir receta [Numero de id]"

Para que lea las ingredientes > "Leer ingredientes"

Para parar la lectura > “Alto”

Para que lea un paso > "Leer paso [Numero de paso]"

Para que muestre un paso > "Abrir paso [numero de paso]"

Para que abra un paso > "Abrir paso [Numero de paso]"

Para ver los ingredientes > "Ver ingredientes"
